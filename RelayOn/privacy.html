개인정보처리방침

최종 수정일: 2025년 2월 11일

Mobeatodam(이하 "개발자")이 제공하는 RelayOn(이하 "앱")은 사용자의 개인정보 보호를 매우 중요하게 생각합니다. 본 개인정보처리방침은 앱이 수집하는 정보와 그 사용 방법을 설명합니다.

1. 수집하는 정보

RelayOn은 다음 정보를 기기 내에서만(로컬) 처리합니다:

  • 알림 데이터: 앱 이름, 패키지명, 알림 제목, 알림 내용, 수신 시간
  • 규칙 설정: 사용자가 생성한 필터링 규칙 및 액션 설정
  • 실행 이력: 액션 실행 결과 (성공/실패 여부)

2. 권한 사용 목적

  • 알림 접근 권한 (NotificationListenerService): 안드로이드 알림을 읽어 사용자가 설정한 규칙에 따라 처리하기 위해 사용됩니다.
  • SMS 발송 권한 (SEND_SMS): 사용자가 설정한 규칙에 따라 알림 내용을 SMS로 전달하기 위해 사용됩니다.
  • 진동 권한 (VIBRATE): 특정 알림 수신 시 기기를 진동시키기 위해 사용됩니다.
  • 인터넷 권한 (INTERNET): 텔레그램 봇을 통해 메시지를 전달하기 위해 사용됩니다.

3. 데이터 저장 및 전송

  • 모든 데이터는 사용자의 기기 내 로컬 데이터베이스에만 저장됩니다.
  • 앱은 개발자의 서버로 어떠한 데이터도 수집하거나 전송하지 않습니다.
  • 사용자가 텔레그램 전달 기능을 활성화한 경우, 해당 알림 내용이 텔레그램 API를 통해 사용자가 지정한 채팅방으로 전송됩니다. 이는 사용자의 명시적인 설정에 의해서만 이루어집니다.
  • 사용자가 SMS 전달 기능을 활성화한 경우, 해당 알림 내용이 사용자가 지정한 전화번호로 SMS 발송됩니다.

4. 제3자 제공

RelayOn은 사용자의 데이터를 제3자에게 판매, 공유, 또는 제공하지 않습니다.

5. 데이터 보관 및 삭제

  • 알림 데이터 및 실행 이력은 앱 내에서 사용자가 직접 삭제할 수 있습니다.
  • 앱을 삭제하면 기기에 저장된 모든 데이터가 자동으로 삭제됩니다.

6. 아동의 개인정보

RelayOn은 13세 미만 아동의 개인정보를 수집하지 않습니다. 13세 미만의 아동이 앱을 사용하는 것을 인지한 경우, 해당 데이터를 즉시 삭제하겠습니다.

7. 개인정보처리방침 변경

본 개인정보처리방침은 변경될 수 있으며, 변경 시 본 페이지를 통해 공지합니다.
