Key Features

📋

Keyword Filtering

Precisely select notifications by app name, title, or text. Supports include/exclude conditions and advanced text patterns.

💬

SMS Forwarding

Automatically forward important alerts via SMS so you can receive them without mobile data.

✈️

Telegram Integration

Instantly send notifications through your Telegram Bot. Configure globally or per-rule.

📳

Custom Vibration

Set custom vibration patterns for specific alerts so you notice them immediately.

🚫

Auto-Dismiss

Automatically clear processed notifications to keep your status bar clean.

📊

Action History

Store and view logs of all captured notifications and executed actions.

Popular Use Cases

🧹 Spam Cleaner
"I want to delete annoying ads instantly, but keep them in history just in case."
  • Trigger: Shopping App + Include: [Ad] Exclude: Order
  • Action: Cancel Notification Save on Device
🚀 Emergency Forwarding
"Forward 'Server Down' or 'Error' alerts from any app directly to our team's Telegram."
  • Trigger: Any App + Text: Error, Down
  • Action: Send to Telegram (Custom Bot/Chat ID)
📳 Unmissable VIP Alert
"I keep my phone on silent, but I need to know immediately when my boss sends a message."
  • Trigger: Messenger App + Text: Boss's Name
  • Action: Custom Vibrate (Max intensity SOS pattern)
💾 Ephemeral Archive
"Banking alerts disappear forever when I accidentally tap them. I want to keep the text safe."
  • Trigger: Banking App
  • Action: Save on Device (Review permanently in History)

How It Works

1

Create Rules

Define rules for which notifications to detect and what actions to execute.

2

Monitor Notifications

The Android NotificationListenerService monitors your alerts in real-time.

3

Auto-Execute

When a rule matches, configured actions like SMS or Telegram forwarding run automatically.

RelayOn은 안드로이드 알림을 감지하여
SMS, 텔레그램 등 원하는 채널로 자동 전달합니다.

💾 기기에 보관

주요 기능

📋

키워드 필터링

앱 이름, 제목, 내용으로 원하는 알림만 정확히 선별합니다. 포함/제외 조건과 고급 텍스트 패턴을 지원합니다.

💬

SMS 자동 전달

중요한 알림을 SMS로 자동 전달하여 데이터 없이도 수신할 수 있습니다.

✈️

텔레그램 전달

텔레그램 봇을 통해 알림을 즉시 전달합니다. 글로벌 설정 또는 규칙별 설정이 가능합니다.

📳

커스텀 진동

특정 알림에 대해 맞춤 진동 패턴을 설정하여 즉시 인지할 정 수 있습니다.

🚫

알림 자동 취소

처리된 알림을 자동으로 제거하여 상태바를 깔끔하게 유지합니다.

📊

이력 관리

모든 알림과 액션 실행 기록을 저장하고 조회할 수 있습니다.

스마트한 활용 사례

🧹 스팸 알림 청소기
"매일 오는 광고 알림은 안 보고 싶지만, 어떤 알림이 왔었는지 나중에 확인은 하고 싶어요."
  • 조건: 쇼핑 앱 + 포함: [광고] 제외: 배송
  • 액션: 알림 즉시 삭제 기기에 조용히 보관
🚀 핵심 키워드 릴레이
"업무용 앱에서 '긴급', '에러' 단어가 포함된 알림이 오면 팀 텔레그램으로 바로 넘기고 싶어요."
  • 조건: 모든 앱 + 내용: 긴급, Error
  • 액션: 텔레그램 전송 (팀 공용 봇/채팅방 지정)
📳 놓칠 수 없는 VIP 진동
"회의 중 폰이 무음이어도, 상사님의 연락은 주머니 속에서 특별한 진동으로 알아채고 싶어요."
  • 조건: 카카오톡 + 제목: 대표님
  • 액션: 커스텀 진동 (크고 강력한 4연속 SOS 진동)
💾 휘발성 알림 영구 보관
"토스 입출금 알림을 실수로 눌러버리면 증발하는데, 알림 원문을 영구적으로 보관할 수 없나요?"
  • 조건: 토스 / 금융 앱
  • 액션: 기기에 안전하게 보관 (히스토리 탭에서 언제든 조회 가능)

어떻게 동작하나요?

1

규칙 생성

어떤 알림을 감지할지, 어떤 액션을 실행할지 규칙을 정의합니다.

2

알림 감지

안드로이드 NotificationListenerService가 실시간으로 알림을 모니터링합니다.

3

자동 실행

규칙에 매칭되면 SMS 전송, 텔레그램 전달 등 설정된 액션이 자동으로 실행됩니다.
